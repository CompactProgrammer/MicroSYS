===== SYSFS Structure =====
Boot block
Volume information
Data area
Folder tree

===== Boot Block =====
The boot block is two sectors mainly used for boot code
===== Volume Information =====
Byte    Size    Contains
0x00    2       SYSFS signature ('FS')
0x02    2       Bytes per sector
0x04    1       Sectors per block
0x05    1       Disk type
0x06    2       Sectors in volume
0x08    2       Sectors per track
0x0a    2       Number of heads
0x0c    4       Volume ID
0x10    1       Size of each folder tree entry
0x12    15      Volume label
===== Data Area =====
This contains all the data in each file
===== Folder Tree (FT) =====
--- File Entry ---
Byte    Size    Contains
0x00    1       Entry type (0x01 for file)
0x01    11      Filename
0x0c    4       File extension
0x10    1       File flags
0x11    1       Creation year
0x12    4       Creation time/date
0x16    4       Modification time/date
0x1a    2       First block of data area
0x1c    4       File size
--- Deleted file entry ---
0x00    1       Entry type (0xde for deleted file)
0x01    11      Filename
0x0c    4       File extension
0x10    1       File flags
0x11    1       Creation year
0x12    4       Creation time/date
0x16    4       Modification time/date
0x1a    2       First block of data area
0x1c    4       File size
