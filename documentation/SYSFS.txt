===== SYSFS Structure =====
Boot block
Volume information
Extended file table
Root directory
Data area

===== Boot Block =====
The boot block is two sectors mainly used for boot code
===== Volume Information =====
Byte    Size    Contains
0x00    2       SYSFS signature ('FS')
0x02    2       Bytes per sector
0x04    1       Sectors per block
0x05    1       Disk type
0x06    2       Sectors in volume
0x08    2       Sectors per track
0x0a    2       Number of heads
0x0c    4       Volume ID
0x10    1       Number of entries per directory
0x11    1       Size of extended file table
0x12    14      Volume label
===== Extended File Table (EFT) =====
These are used if blocks of a file aren't consecutive
Entry 0 is unused
Byte    Size    Contains
0x00    2       First block in data area
0x02    2       Last block in data area
===== Root Directory =====
--- File Entry ---
Byte    Size    Contains
0x00    1       Entry type (0x01 for file)
0x01    11      Filename
0x0c    4       File extension
0x10    1       File flags
0x11    1       Creation year
0x12    4       Creation time/date
0x16    2       Modification date
0x18    2       First block in data area
0x1a    2       First EFT entry (0 if no EFT entry)
0x1c    4       File size
--- Deleted file entry ---
0x00    1       Entry type (0xde for deleted file)
0x01    11      Filename
0x0c    4       File extension
0x10    1       File flags
0x11    1       Creation year
0x12    4       Creation time/date
0x16    4       Modification time/date
0x1a    2       First block of data area
0x1c    4       File size
===== Data Area =====
This contains all the data in each file
