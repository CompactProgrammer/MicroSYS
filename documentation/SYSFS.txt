===== SYSFS Structure =====
Boot block
Root folder
Data area

===== Boot Block =====
The boot block is two sectors mainly used for boot code
Also contains the volume information at the very end of the block
--- Volume Information ---
Byte    Size    Contains
0x00    2       SYSFS signature ('FS')
0x02    2       Bytes per sector
0x04    1       Sectors per block
0x05    1       Disk type
0x06    2       Sectors in volume
0x08    2       Sectors per track
0x0a    2       Number of heads
0x0c    4       Volume ID
0x10    2       Sectors in root folder
0x12    2       Reserved
0x14    12      Volume label
--- Disk Types ---
Byte    Type
0xf2    1.2MB floppy
0xf3    360K floppy
0xf4    1.44MB floppy
0xf7    720K floppy
0xf8    2.88MB floppy
===== Root Directory =====
--- Entry ---
Byte    Size    Contains
0x00    11      Filename
0x0b    3       File extension
0x0e    1       Entry flags
0x0f    1       Creation year
0x10    4       Creation time/date
0x14    2       Modification date
0x16    1       Modification year
0x17    1       Reserved
0x18    2       First block of file
0x1a    2       Last block of file
0x1c    4       File size
--- Entry Flags ---
Bit Flag
0   Read-only
1   Hidden
2   System
3   Backed up
===== Data Area =====
This contains all the data in each file
