Module header:
Byte    Size    Contains
0x00    1       Header version
0x01    1       Module type
0x02    2       Memory required
0x04    2       Code pointer
0x06    2       Reserved
0x08    24      Module name

Module types:
Byte    Type
0x00    Reserved
0x01    Filesystem
0x02    Video adapter
0x03    Storage drive
0xff    Misc. module
