MicroSYS Function 0x03 - Debugger Output

===== Expects =====
AH      0x03
AL      Byte to output
===== Returns =====
None
===== Info =====
Sends a character to any possible debug ports.
For example, this function will write to port 0xe9, which prints to the Bochs terminal.
WARNING! On a real PC, these ports could be used for something other than debugging. Use carefully!
