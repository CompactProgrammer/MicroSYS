Call INT 0x21 to call these functions

===== Error codes =====
AH      Error
0x00    No error
0x01    Invalid function number
0x02    File not found
0x03    Path not found
0x04    Too many open files
0x05    INT 0x13 error (error code in AL)
0x06    File/folder access denied
0x07    File already exists
0x08    Invalid file format
0xff    Other error

===== Functions =====
AH      Function
0x00    Terminate
0x01    Get/set INT vector
0x02    Get MicroSYS version
0x03    Load child program
0x04    Execute child program
0x05    Get/set environment variable
0x06    Get error code of last operation

===== IO.SYS Functions =====
0x00    Keyboard input
0x01    Print character
0x02    Check for keyboard input
0x03    Print string
0x04    Clear screen
0x05    Debugger output
0x06    Get/set date
0x07    Get/set time
0x10    Make directory
0x11    Delete directory
0x12    Set current directory
0x13    Create file
0x14    Delete file
0x15    Open file
0x16    Close file
0x17    Get/set file cursor position
0x18    Read file
0x19    Write file
0x1a    Get/set attributes
0x1b    Rename file
0x1c    Move file
0x1d    Get file size
0x1e    Get time & date of file
0x1f    Commit unsaved file data
